Materi Topologi Jaringan

Topologi jaringan adalah cara atau konsep untuk menghubungkan beberapa atau banyak komputer sekaligus menjadi suatu jaringan yang saling terkoneksi. Berikut adalah beberapa jenis topologi jaringan:

Topologi Bus

Topologi ini menggunakan satu kabel utama sebagai media transmisi data yang menghubungkan semua perangkat.

Karakteristik:

  • Menggunakan kabel coaxial
  • Terminator di setiap ujung kabel
  • Mudah diimplementasikan
  • Jika kabel utama putus, seluruh jaringan down

Topologi Ring

Setiap perangkat terhubung ke dua perangkat lainnya, membentuk jalur melingkar (cincin).

Karakteristik:

  • Data mengalir dalam satu arah
  • Menggunakan token passing
  • Lebih teratur dari topologi bus
  • Jika satu node rusak, seluruh jaringan terganggu

Topologi Star

Semua perangkat terhubung ke satu titik pusat (switch atau hub).

Karakteristik:

  • Paling populer digunakan
  • Mudah dikelola dan di-troubleshoot
  • Jika satu node rusak, tidak mempengaruhi node lain
  • Jika hub/switch rusak, seluruh jaringan down

Topologi Mesh

Setiap perangkat terhubung langsung dengan semua perangkat lainnya.

Karakteristik:

  • Redundansi tinggi
  • Keamanan baik
  • Membutuhkan banyak kabel
  • Biaya implementasi tinggi

Topologi Tree

Kombinasi dari topologi star dan bus, dengan hierarki yang jelas.

Karakteristik:

  • Mudah untuk ekspansi
  • Pengelolaan lebih mudah
  • Jika backbone rusak, seluruh jaringan down
  • Lebih banyak kabel dibutuhkan

Topologi Hybrid

Kombinasi dari dua atau lebih topologi jaringan.

Karakteristik:

  • Fleksibel dan dapat disesuaikan
  • Menggabungkan kelebihan berbagai topologi
  • Instalasi dan konfigurasi kompleks
  • Biaya relatif tinggi

Tentang Aplikasi

Aplikasi ini dikembangkan sebagai media pembelajaran Informatika Kelas XI, membantu siswa memahami topologi jaringan secara visual dan interaktif.

Tujuan Pembelajaran

Setelah menggunakan media pembelajaran ini, siswa diharapkan mampu:

  • Memahami berbagai jenis topologi jaringan
  • Menganalisis kelebihan dan kekurangan masing-masing topologi
  • Memvisualisasikan cara kerja berbagai topologi jaringan
  • Menerapkan pengetahuan dalam merancang jaringan sederhana

Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan HTML5, CSS3, dan JavaScript dengan Canvas API untuk simulasi interaktif. Desain responsif memastikan pengalaman belajar yang optimal di berbagai perangkat.
